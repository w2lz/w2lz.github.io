# P170. 包含重复元素的数组排列


<!--more-->

## 描述

这个题目说的是，给你一个整数数组，数组中可能包含重复元素，你要返回这个数组所有不同的排列。

```markdown
比如说，给你的数组是：

[0, 1, 0]

由 [0, 1, 0] 构成的不同排列只有 3 种，分别是：

[0, 0, 1]
[0, 1, 0]
[1, 0, 0]
```
