# P216. 数字组合


<!--more-->

## 描述

这个题目说的是，给你两个正整数 n 和 k，你要从 1 ~ n 中取 k 个数字，并返回所有可能的数字组合。

```markdown
比如说，给你的 n 和 k 分别是：

n = 4
k = 2

你要从 1 ~ 4 中取 2 个数字，一共有 6 种可能的组合：

[1, 2]
[1, 3]
[1, 4]
[2, 3]
[2, 4]
[3, 4]
```
