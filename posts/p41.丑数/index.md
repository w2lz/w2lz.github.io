# P41. 丑数


<!--more-->

## 描述

这个题目说的是，给你一个数字，你要判断它是不是一个丑数。丑数的定义是质因数只包含 2，3，5 的正整数。另外，1 作为特例，也定义为丑数。

```markdown
比如说给你的数字是 45，45 做质因数分解，可以写成：

45 = 3 x 3 x 5

不包含 2，3，5 以外的质因数，因此它是一个丑数。

再比如说 42，它做质因数分解得到：

42 = 2 x 3 x 7

7 不在 2，3，5 中，因此 42 不是丑数。
```
