# P99. 第 N 个丑数


<!--more-->

## 描述

这个题目说的是，给你一个整数 n，你要返回第 n 个丑数。丑数的定义是质因数只包含 2，3，5 的正整数。另外，1 作为特例，也定义为丑数。

```markdown
比如说，给你的 n=6。前 6 个丑数是：

1, 2, 3, 4, 5, 6 // 它们的质因数都只包含 2，3，5

于是你要返回第 6 个丑数 6。
```
