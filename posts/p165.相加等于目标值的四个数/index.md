# P165. 相加等于目标值的四个数


<!--more-->

## 描述

这个题目说的是，给你一个整数数组和一个目标值，你要找到数组中四个数相加等于目标值的所有可能组合。返回的答案里，每个组合都是唯一的，不能重复。

```markdown
比如说，给你的数组是：

[1, 0, -1, 0, -3, 2]

给你的目标值是 0。

这个数组中有两个组合，使得 4 个数相加等于目标值 0。第一个组合是：

[-3, 0, 1, 2]

虽然数组中有两个 0，但是不管使用哪个 0，组合 [-3, 0, 1, 2] 都只算作一个。

另一个相加等于目标值的组合是：

[-1, 0, 0, 1]
```
