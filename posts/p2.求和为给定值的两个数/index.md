# P2. 求和为给定值的两个数


<!--more-->

## 描述

这个题目说的是，给你一个整数数组和一个目标值，你要找到数组里两个整数，它们的和等于目标值。然后返回这两个整数的下标。

```markdown
比如说给你的整数数组是：

1, 2, 3, 6, 8, 11

目标值是 10。那么，满足条件的两个整数是，2 和 8，它们的和是 10。所以你要返回它们的下标是 1 和 4。
```
