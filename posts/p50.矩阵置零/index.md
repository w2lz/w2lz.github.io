# P50. 矩阵置零


<!--more-->

## 描述

这个题目说的是，给你一个 m x n 的矩阵，你要把这个矩阵中等于 0 的元素所在的行和列都置 0。

```markdown
比如说，给你的矩阵 a 是：

 1, 2, 3
 4, 0, 6
 0, 8, 9

这个矩阵中有两个 0，把它们所在的行和列都置 0 后，得到的矩阵是：

 0, 0, 3
 0, 0, 0
 0, 0, 0
```
