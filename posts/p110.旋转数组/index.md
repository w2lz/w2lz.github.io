# P110. 旋转数组


<!--more-->

## 描述

这个题目说的是，给你一个数组和一个数字 k，你要把数组右边的数字旋转到数组左边，一次旋转一个数字，共旋转 k 次。

```markdown
比如说，给你的数组是：

0, 1, 2, 4, 8

给你的数字 k 是 3：

k = 3

把数组右边的数字一个个旋转到左边，操作 3 次。先是 8，然后 4，最后是 2。剩下的 0 和 1 保持不动。最后得到旋转后的数组是：

2, 4, 8, 0, 1
```
