# P56. 有序数组中的单身数字


<!--more-->

## 描述

这个题目说的是，给你一个排好序的整数数组，里面的数字都出现两次，只有一个数字出现了一次，我们管它叫单身数字，你要写代码找到这个单身数字。

```markdown
比如说给你的有序数组是：

1, 1, 2, 2, 4, 4, 6, 8, 8

这个数组里 6 只出现了一次，因此你要返回的数字就是 6。
```
