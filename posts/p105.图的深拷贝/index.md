# P105. 图的深拷贝


<!--more-->

## 描述

这个题目说的是，给你一个无向图，你要返回这个图的深拷贝。

```markdown
比如说，给你的图是：

    1 --- 2
   / \   /
  /   \ /
 4 --- 8

你要返回这个图的一份深拷贝，为了和原图区分，我们记为：

    1’--- 2’
   / \   /
  /   \ /
 4’--- 8’

每个图节点可以表示成它的节点值以及它相邻的节点。比如这个图，可以表示成：

 1: 2, 4, 8
 2: 1, 8
 4: 1, 8
 8: 1, 2, 4

拷贝图本质上就是把这组结构再复制一遍即可。
```
