# P229. 二叉树的右视图


<!--more-->

## 描述

这个题目说的是，给你一棵二叉树，并且你站在这棵树的右边，你要返回从上到下看到的节点值。

```markdown
比如说，给你的二叉树是：

     1
   /   \
  2     4
 / \
6   8

站在这棵二叉树的右边看过来，从上到下看到的数字依次是：

[1, 4, 8]
```
