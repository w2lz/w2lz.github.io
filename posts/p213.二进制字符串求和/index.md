# P213. 二进制字符串求和


<!--more-->

## 描述

这个题目说的是，给你两个用字符串表示的二进制数字，你要返回它们的和，并且和的形式同样为二进制字符串。

其中，给你的两个字符串都不为空，并且只包含 0/1 字符。

```markdown
比如说，给你的两个字符串是：

"1101"
 "111"

它们求和后的二进制字符串是 "10100"。
```
