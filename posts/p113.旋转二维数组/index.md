# P113. 旋转二维数组


<!--more-->

## 描述

这个题目说的是，给你一个 n x n 的二维数组，你要沿顺时针方向将它旋转 90 度。要求你不能使用额外的存储空间，就地在原数组操作。

```markdown
比如说，给你的二维数组是：

1, 2, 3
4, 5, 6
7, 8, 9

顺时针旋转 90 度后，得到的二维数组是：

7, 4, 1
8, 5, 2
9, 6, 3
```
