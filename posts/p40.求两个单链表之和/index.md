# P40. 求两个单链表之和


<!--more-->

## 描述

这个题目说的是，给你两个非空的单链表，它们代表两个非负整数，并且逆序表示。你要将这两个数求和，并将结果以链表形式返回。你不需要考虑前导 0 这种情况，也就说 3 不会表示成 003 这样子。

```markdown
比如说给你的两个链接表是：

1 -> 2 -> 3
6 -> 7 -> 8 -> 9

1 -> 2 -> 3 表示的整数是 321，6 -> 7 -> 8 -> 9 表示的整数是 9876。我们需要输出的是它们求和后的链表：

7 -> 9 -> 1 -> 0 -> 1
```
