# P238. 能否参加所有会议


<!--more-->

## 描述

这个题目说的是，给你一系列的会议时间区间，每个时间区间由开始时间和结束时间构成。你要判断出同一个人是否可以参加所有会议。

```markdown
比如说，给你两个时间区间：

[2, 4]
[4, 6]

这两个时间区间没有重叠，因此所有会议都能参加，返回 true。

如果再增加一个会议，得到三个时间区间：

[2, 4]
[4, 6]
[1, 3]

由于 [1, 3] 和 [2, 4] 在时间上有重叠，因此同一个人不可能参加所有会议，返回 false。
```
