# P176. 两个数组的交集


<!--more-->

## 描述

这个题目说的是，给你两个非空数组，你要实现一个函数，来计算它们的交集。

注意，交集中元素的顺序不重要，以什么顺序返回都可以。

```markdown
比如说，给你的第一个数组是：

1, 4, 4, 2

第二个数组为：

4, 8, 4, 4

这两个数组都包含两个 4，因此它们的交集为：

4, 4
```
