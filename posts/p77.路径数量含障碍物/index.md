# P77. 路径数量（含障碍物）


<!--more-->

## 描述

这个题目说的是，给你一个矩阵，矩阵元素有 0 和 1 两种值。0 表示矩阵中这个格子是空的，可以走；1 表示这个格子里有障碍物，不能走。你在这个矩阵中只能向右或向下走，那么从矩阵左上角走到右下角，共有多少条不同的路径。

```markdown
比如说给你的矩阵是：

 0, 0, 1, 0
 0, 0, 0, 0

第 0 行的 1 表示那里有个障碍物，不能走。从这个矩阵的左上角走到右下角，共有 2 条不同的路径，因此要返回 2。
```
