# P112. 求和为给定值的组合


<!--more-->

## 描述

这个题目说的是，给你一个正整数数组，数组中不包含重复元素，同时给你一个正整数目标值，你要找到数组中和为目标值的所有组合。另外，数组中每个元素都可以使用无限多次，并且答案中不能包含重复组合。

```markdown
比如说，给你的数组是：

4, 2, 8

给你的目标值是 6。数组中和为 6 的组合有：

[4, 2]
[2, 2, 2]
```
