# P79. 最长递增子序列的长度


<!--more-->

## 描述

这个题目说的是，给你一个整数数组，你要计算数组里最长递增子序列的长度。其中，子序列不要求连续。

```markdown
比如说，给你的数组 a 是：

1, 8, 2, 6, 4, 5

在这个数组里，最长的递增子序列是：

1, 2, 4, 5

因此你要返回它的长度 4。
```
