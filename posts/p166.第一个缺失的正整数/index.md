# P166. 第一个缺失的正整数


<!--more-->

## 描述

这个题目说的是，给你一个整数数组，你要找到数组中没有出现的最小正整数。

这个题目要求算法只能使用 O(n) 的时间，以及 O(1) 的辅助空间。

```markdown
比如说，给你的数组是：

[2, -1, 4, 1, 8]

我们可以从最小的正整数 1 开始，检查第一个没出现在这个数组的正整数。1 和 2 都出现了，而 3 不在这个数组中，因此第一个缺失的正整数是 3。
```
