# P142. 除了自身元素的数组乘积


<!--more-->

## 描述

这个题目说的是，给你一个大小为 n 的整数数组 nums，其中 n > 1。你要返回一个数组 output。其中，output(i) 是原数组中除了 nums(i) 以外，所有数字的乘积。注意，这道题目的求解过程不能使用除法。

```markdown
比如说，给你的数组是：

1, 4, 2, 8

你要返回的数组就是：

64, 16, 32, 8

可以看到返回的数组中，每个位置上的数字都是原数组中除了那个位置以外的所有数字乘积。
```
