# P95. 最长连续整数序列的长度


<!--more-->

## 描述

这个题目说的是，给你一个无序的整数数组，你要找到数组中元素能够组成的最长连续整数序列，然后返回它的长度。

比如说，给你的数组是 8, 4, 2, 1, 2, 3, 6，这里面的数字能够组成的最长连续整数序列是 1, 2, 3, 4，因此你要返回它的长度 4。
