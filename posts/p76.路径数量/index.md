# P76. 路径数量


<!--more-->

## 描述

这个题目说的是，给你两个整数 m 和 n，表示的是一个 m x n 的矩阵，任何时候你只能向右或向下移动，那么从矩阵左上角移动到右下角，共有多少条不同的路径。

```markdown
比如说，给你的 m = 2，n = 4。简单地画出这个矩阵，使用 0 作为占位符：

0, 0, 0, 0
0, 0, 0, 0

从这个矩阵的左上角走到右下角，共有 4 条不同的路径，因此返回 4。
```
