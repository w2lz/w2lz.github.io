# P193. 二叉树的 Z 字形遍历


<!--more-->

## 描述

这个题目说的是，给你一棵二叉树，要求你从根节点到叶子节点一层一层地进行 Z 字形遍历，也就是先从左向右访问一层节点，然后从右向左访问下一层节点。以这样的方式交替去访问二叉树上每一层节点，并且将访问的结果以二维数组的形式返回。

```markdown
比如说，给你的二叉树是：

  1
 / \
2   4
   / \
  8  16

它的 Z 字形遍历结果是：

[
 [1],
 [4, 2],
 [8, 16]
]
```
