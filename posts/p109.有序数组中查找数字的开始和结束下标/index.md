# P109. 有序数组中查找数字的开始和结束下标


<!--more-->

## 描述

这个题目说的是，给你一个递增排序的数组和一个目标值，你要找到目标值在这个数组中的开始下标和结束下标。如果找不到目标值，就返回 [-1, -1]。

```markdown
比如说，给你的递增数组是：

1, 2, 2, 4, 4, 8, 8

给你的目标值是 2。2 在这个数组中的开始下标是 1，结束下标是 2，于是你要返回：

[1, 2]

如果给你的目标值是 0，0 不在这个数组中，因此你要返回：

[-1, -1]
```
