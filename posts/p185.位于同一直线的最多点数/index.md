# P185. 位于同一直线的最多点数


<!--more-->

## 描述

这个题目说的是，给你二维平面上 n 个点，你要计算出最多有多少个点可以位于同一条直线上。

注意，给你的 n 个点中，可能包含相同的点。

```markdown
比如说，给你的点是：

[[0,0], [0,0], [0,1], [1,1], [2,2]]

两个 [0,0] 和 [1,1]、[2,2] 这 4 个点都位于 y = x 这条直线上。并且不存在其他直线，可以包含更多的点。因此位于同一直线上最多的点数就为 4。
```
